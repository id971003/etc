"X") X 에 따라 string 값 표시를 다르게해준다

F  : 소숫점에서 짤라버림 
ex 123456.789
	F0 = 123456
	F1 = 123456.7
	F2 = 123456.78
	F3 = 123456.789

N : 소숫점에서 짜르고 1000단위 쉼표 찍음
ex 123456.789	
	N1 = 123,456.7
	N2 = 123,456.78
	N3 = 123,456.789

P : 백분률 
ex 0.345
	P0 = 30%
	P1 = 30.4%
	P2 = 30.45%

D : 0을 자릿수만큼 넣음
ex 123
	D0 = 123
	D1 = 0123
	D2 = 00123

$"{value:0.#}";
소수점 1번째 자리가 0으로 떨어지면 표기하지않음.
